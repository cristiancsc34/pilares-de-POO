Herança em POO

O que é Herança?

Herança é um dos pilares da Programação Orientada a Objetos (POO) que permite criar uma nova classe com base em uma classe existente. A classe filha herda atributos e métodos da classe pai, promovendo o reaproveitamento de código e facilitando a manutenção. Além disso, a herança possibilita a especialização de comportamentos, onde a classe filha pode adicionar ou modificar funcionalidades da classe pai.

Com a herança, é possível criar uma hierarquia entre as classes, onde uma classe mais genérica pode ser estendida por classes mais específicas, tornando o sistema mais modular e flexível.

Exemplo de Código de Herança em POO

class Animal:
    def __init__(self, nome: str):
        self.nome = nome

    def falar(self):
        pass

class Cachorro(Animal):
    def falar(self):
        return f"{self.nome} diz: Au Au!"

class Gato(Animal):
    def falar(self):
        return f"{self.nome} diz: Miau!"

# Criando objetos das classes filhas
cachorro = Cachorro("Rex")
gato = Gato("Felix")

# Chamando o método falar das classes filhas
print(cachorro.falar())
print(gato.falar())

Explicação do Código

Classe Animal:
A classe `Animal` é a classe pai, que define o comportamento básico dos animais com o atributo `nome` e o método `falar`, mas não implementa o comportamento de falar. O método `falar` é deixado em branco para ser implementado pelas classes filhas.

Classes Cachorro e Gato:
As classes `Cachorro` e `Gato` herdam de `Animal`. Elas implementam o método `falar` de maneira específica para cada tipo de animal. Cada uma delas define um comportamento único para o método `falar` que retorna uma mensagem diferente, como o latido do cachorro ou o miado do gato.

Benefícios da Herança:
A herança permite que as classes `Cachorro` e `Gato` reutilizem o código da classe `Animal`, sem precisar duplicá-lo. Se houver algum comportamento comum entre os animais, como o nome, a classe `Animal` pode ser compartilhada. Isso torna o código mais limpo, modular e fácil de estender no futuro, caso seja necessário adicionar novos tipos de animais.
