Abstração em POO

O que é Abstração?

Abstração é um dos pilares da Programação Orientada a Objetos (POO). Ela envolve ocultar os detalhes de implementação complexos e expor apenas o que é essencial para o usuário ou outras partes do sistema. Em outras palavras, abstração permite que você defina interfaces simples e claras, enquanto o funcionamento interno pode ser escondido.

Isso é feito através da definição de classes e métodos abstratos, que especificam o comportamento esperado sem detalhar como ele será implementado. Em POO, a abstração permite criar código mais modular e reutilizável, simplificando a interação com objetos e promovendo uma maior organização.

Exemplo de Código de Abstração em POO

from abc import ABC, abstractmethod

# Classe Abstrata
class Animal(ABC):
    @abstractmethod
    def emitir_som(self):
        pass

# Subclasses que implementam o método abstrato
class Cachorro(Animal):
    def emitir_som(self):
        return "Au Au!"

class Gato(Animal):
    def emitir_som(self):
        return "Miau!"

# Função que demonstra o uso da abstração
def fazer_animal_falar(animal: Animal):
    print(animal.emitir_som())

# Criando instâncias de Cachorro e Gato
cachorro = Cachorro()
gato = Gato()

# Chamando a função de abstração
fazer_animal_falar(cachorro)  # Saída: Au Au!
fazer_animal_falar(gato)      # Saída: Miau!

Explicação do Código

Classe Abstrata Animal:
A classe Animal é definida como uma classe abstrata usando `ABC` (Abstract Base Class). O método `emitir_som` é declarado como abstrato com o decorador `@abstractmethod`. Isso significa que ele não possui implementação na classe base e deve ser implementado pelas subclasses.

Subclasses Cachorro e Gato:
As classes `Cachorro` e `Gato` herdam de `Animal` e implementam o método `emitir_som`. Cada uma tem seu próprio comportamento ao emitir um som (o latido do cachorro e o miado do gato).

Função `fazer_animal_falar`:
A função `fazer_animal_falar` aceita qualquer objeto do tipo `Animal` e chama o método `emitir_som()`. Isso demonstra a abstração de que, independentemente do tipo de animal (cachorro ou gato), o comportamento de emitir som é garantido pela classe abstrata.
